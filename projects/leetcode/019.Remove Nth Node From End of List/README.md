# 删除链表的倒数第N个节点(Remove Nth Node From End of List)

## Description

给定一个链表, 删除链表的倒数第 n 个节点, 并且返回链表的头节点

示例：
```
给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.
```
